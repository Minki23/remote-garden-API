Views
=====

The **Views** module contains **FastAPI route handlers** responsible for 
rendering HTML pages and serving templates.

Structure
---------

- ``common_view.py`` – shared view helpers.
- ``main_view.py`` – main page rendering.
- ``__init__.py`` – module initializer.
